GooeyStat
=========

About 
----- 

GooeyStat is an application for basic statistical analysis.  It started as a
student project in a Statistical Methods course at `Wartburg College`_.  The
goal is to have a Open Source, cross-platform program for doing the
calculations of an introductory statistics class.

.. _`Wartburg College`: http://www.wartburg.edu

Dependencies
------------

In addition to Python, GooeyStat requires a number of additional libraries.
You will need

  - SciPy_ (tested with version 0.7)
  - PyQT_ (version 4 or higher)

.. _SciPy: http://www.scipy.org
.. _PyQT: http://www.riverbankcomputing.co.uk/software/pyqt/intro

In many cases, these dependencies will be provided by your distribution.
Otherwise, they can be downloaded and installed from the above locations.

Installation
------------

Assuming that the above dependencies are satisfied, GooeyStat can be installed
by running::

  python setup.py install

In some cases you may need root permissions to run this command.
